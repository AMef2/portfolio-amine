## Projets

### SeatFindr: Suivi d'occupation en temps réel
Système IoT complet pour suivre la disponibilité des salles d'étude. Capteur radar mmWave (HLK-LD2450) sur ESP32, Firebase en backend, application Android native.

**Réalisations :** Architecture du firmware C++, optimisation de la communication basse latence vers Firebase, interface d'administration, et simulation Python pour le réglage des paramètres de détection.

`C++` `ESP32` `Firebase` `Java (Android)` `Python`

### Campus Events & Ticketing
Plateforme web de gestion d'événements universitaires avec génération de QR codes pour les check-ins.

**Réalisations :** Développement full-stack (React/Express), authentification Firebase, gestion des flux d'inscription.

`React.js` `Express.js` `Node.js` `Firebase`

### Hovercraft — ENGR 290
Aéroglisseur autonome. Système Arduino, IMU pour la stabilité active, contrôle de portance/poussée.

`Arduino` `C++` `IMU`

### Auto École Jean — Gestion administrative
Agent de bureau. Transition des dossiers papier vers le format numérique, structuration de données sous Excel, traitement des paiements et attestations SAAQ.

`Excel` `Data Management` `Customer Service`
